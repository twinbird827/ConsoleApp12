ConsoleApp12

## Overview

ボールカウント(野球) https://qiita.com/Nabetani/items/ebd8a56b41711ba459f9

## 問題

<p><p>野球のボールカウント・アウトカウントの遷移を計算する。（得点・ランナー・イニング の計算は不要）<br>
<br />ただし、ストライク・ボール・ファウル・ヒット・ピッチャーフライしかない。<br>
<br />細かいルールは下記の通り：</p></p>

<p><ul>
<br /><li>ストライクが３つになったらアウトが増え、ストライクとボールがゼロになる。</li>
<br /><li>ボールが4つになったらフォアボールになり、ストライクとボールがゼロになる。アウトは増えない。</li>
<br /><li>ヒットを打ったらストライクとボールがゼロになる。アウトは増えない。</li>
<br /><li>ピッチャーフライを打ったらストライクとボールがゼロになり、アウトが増える。</li>
<br /><li>アウトが3つになったら、アウト・ストライク・ボール全てゼロになる。</li>
<br /><li>ファウルの場合、もともとストライクが1以下の場合はストライクが増え、ストライクが2の場合には変化なし。</li>
<br /><li>入力は "sbsfbhsshssbbffbbssbs" のように、ひとつながりの文字列として与えられる。</li>
<br /><li>s, b, f, h, p がそれぞれ ストライク、ボール、ファウル、ヒット、ピッチャーフライ を意味する。</li>
<br /><li>出力は、アウト・ストライク・ボールの順にカウントをつなげたものをコンマで区切る。例を参照。</li>
<br /><li>不正入力には対処しなくてよい。</li>
<br /><li>最終回を超えることも考慮しなくてよい。</li>
<br /></ul></p>

<p><h2>
<br /><span class="fragment"></span><a href="#%E5%85%A5%E5%8A%9B%E3%81%A8%E5%87%BA%E5%8A%9B%E3%81%AE%E4%BE%8B"><i class="fa fa-link"></i></a>入力と出力の例</h2></p>

<p><p>以下、入力 -&gt; 出力 の形式で例を示す。</p></p>

<p><p>s -&gt; 010<br>
<br />sss -&gt; 010,020,100<br>
<br />bbbb -&gt; 001,002,003,000<br>
<br />ssbbbb -&gt; 010,020,021,022,023,000<br>
<br />hsbhfhbh -&gt; 000,010,011,000,010,000,001,000<br>
<br />psbpfpbp -&gt; 100,110,111,200,210,000,001,100<br>
<br />ppp -&gt; 100,200,000<br>
<br />ffffs -&gt; 010,020,020,020,100<br>
<br />ssspfffs -&gt; 010,020,100,200,210,220,220,000<br>
<br />bbbsfbppp -&gt; 001,002,003,013,023,000,100,200,000<br>
<br />sssbbbbsbhsbppp -&gt; 010,020,100,101,102,103,100,110,111,100,110,111,200,000,100<br>
<br />ssffpffssp -&gt; 010,020,020,020,100,110,120,200,210,000</p></p>
